# TuteDudeAssignment4


Overview : 
This project demonstrates basic file handling operations in Python.
It includes two tasks that show how to read from a file, write data to a file, append data, and handle errors gracefully.


Task 1 : focuses on reading the contents of a text file named sample.txt and displaying its contents line by line.

 Functionality :

-> Opens the file sample.txt in read mode.
-> Prints a heading: "Reading file content:"
-> Reads the file line by line.
-> Displays each line with a line number.
-> Handles errors if the file does not exist.


 Concepts Used :

-> File opening and reading
-> Looping through file content
-> Exception handling (FileNotFoundError)
